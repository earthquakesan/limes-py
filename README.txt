============
  LIMES Py
============

LIMES Py provides an interface to LIMES link discovery framework.
it most useful for tasks involving linking CSV files to existing ontologies. 
Typical usage often looks like this::

    #!/usr/bin/env python

    from limespy import CsvLinker

    CsvLinker.link_my_csv(mycsv, dbpedia)
